## 🚀 What is PassFlow?

PassFlow is an open-source platform for deploying **ephemeral AI agents** that reason about your infrastructure, execute real actions, and destroy themselves when done. Ephemeral pods, permanent results.

### Key Features

- ⚡ **< 100ms Cold Start** — From trigger to pod execution in milliseconds
- 🧠 **6 LLM Providers** — Anthropic, OpenAI, Bedrock, Gemini, Azure, OCI with automatic circuit breaker
- 🔧 **27 MCP Tools** — Control the entire platform from Claude Code or Cursor
- 🔒 **100% Isolation** — Each job runs in its own K8s pod with configurable resources

## 🎯 Use Cases

- **Incident Response** — Automated triage and remediation at 3 AM
- **Infrastructure Automation** — Self-governing cloud infrastructure
- **DevOps Workflows** — AI-powered CI/CD and deployment pipelines
